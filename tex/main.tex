\begin{abstract}
We propose a framework for structured sequence reasoning that combines masked diffusion denoising with logical tensor-network inference and adaptive mask policies. Given a partially masked symbolic sequence $x^{(k)} \in (A \cup \{\Mask\})^n$ over a finite domain $A$, we iteratively repair it by (1) a denoising proposal $p_i(v)$, (2) exact logical marginals $q_i(v) = P_\Psi(x_i=v\mid x_\Omega)$ from a tensor-network constraint satisfaction backend, (3) a product-of-experts correction $\tilde p_i(v) \propto p_i(v)^\beta q_i(v)^{1-\beta}$, and (4) an adaptive mask policy that selects which positions to fill or remask. On $4\times4$ Sudoku, TN marginal denoising achieves 100\% completion in 1.4 steps versus 2.6 for local heuristics. The contraction-based tensor backend matches brute-force exact marginals to machine precision ($\|q^C - q^{\text{BF}}\|_\infty < 10^{-14}$). These results establish the building blocks for verifier-guided, tensor-structured reasoning without a large neural model.
\end{abstract}

\section{Introduction}
\label{sec:intro}
% ==============================================================================

Masked diffusion language models (MDLMs) generate sequences by iterative denoising rather than left-to-right autoregressive prediction~\cite{austin2021structured,chang2022masked,savinov2021step}. This makes them natural candidates for globally constrained generation: they can refine arbitrary positions, use bidirectional context, and remask inconsistent spans. However, standard denoising relies solely on learned token statistics and lacks explicit mechanisms for logical consistency.

Tensor networks (TNs) provide complementary tools: compact low-rank representations of high-dimensional discrete states, factor-graph inference over logical constraints, and differentiable proxies for symbolic reasoning~\cite{glasser2020lattice,stoudenmire2016supervised,novikov2015tensorizing}. The connection between diffusion and tensor-network inference has been noted in the context of diffusion models for continuous data~\cite{ho2020denoising}, but the discrete symbolic case remains underexplored.

We propose a framework that explicitly couples a denoising proposal with a logical tensor-network backend. At each diffusion step, the backend computes exact conditional marginals over masked positions by contracting a factor graph encoding the domain's constraints. These marginals are combined with the denoiser's proposal via product-of-experts (PoE):

\begin{equation}
\log \tilde p_i(v) = \beta \log p_i(v) + (1-\beta) \log q_i(v) - \log Z_i.
\label{eq:poe}
\end{equation}

The parameter $\beta \in [0,1]$ interpolates between pure neural denoising ($\beta=1$) and pure logical inference ($\beta=0$).

Our contributions are:
\begin{itemize}[leftmargin=2em]
    \item A formal view of masked diffusion as iterative defect correction, with verifier diagnostics providing local residuals $r_i(x)$ for adaptive remasking.
    \item A tensor-factor contraction backend (variable elimination) that computes exact logical marginals to machine precision, matching brute-force oracle enumeration.
    \item A product-of-experts mechanism for combining neural proposals with logical marginals.
    \item An experimental validation on $4\times4$ Sudoku, showing that TN marginals solve in 1.4 steps versus 2.6 for local heuristics, with both methods achieving 100\% success at 50\% masking.
\end{itemize}

\section{Method}
\label{sec:method}
% ==============================================================================

\subsection{Masked state and constraint satisfaction}
\label{sec:state}

Let $x \in (A \cup \{\Mask\})^n$ be a {\em masked state} over a finite domain $A$ of size $d$, where $\Mask = -1$ denotes an unobserved position and values $0,\dots,d-1$ are valid assignments. The state is governed by a collection of constraint groups $\cG = \{G_1,\dots,G_m\}$, each requiring that the variables in $G$ take pairwise distinct values (all-different constraint). The constraint satisfaction function is

\begin{equation}
\Psi(x) = \prod_{G \in \cG} \psi_G(x_G),
\qquad
\psi_G(x_G) = \mathbf 1\{x_i \neq x_j \text{ for all } i,j \in G,\; i\neq j\}.
\label{eq:psi}
\end{equation}

Global violation and local residuals are defined as

\begin{align}
V(x) &= |\{G \in \cG : \psi_G(x_G) = 0\}|, \\
r_i(x) &= |\{G \in \cG : i \in G \text{ and } \psi_G(x_G) = 0\}|.
\label{eq:residual}
\end{align}

The local residual $r_i(x)$ counts the number of violated groups containing position $i$, providing a signal for verifier-guided remasking.

\subsection{Tensor-factor contraction backend}
\label{sec:backend}

Given observed positions $\Omega \subseteq \{1,\dots,n\}$ with values $x_\Omega$, the conditional marginal for a masked variable $i \notin \Omega$ is

\begin{equation}
q_i(v) = P_\Psi(x_i = v \mid x_\Omega)
= \frac{\sum_{x_{M\setminus\{i\}}} \Psi(x_\Omega, x_i=v, x_{M\setminus\{i\}})}
        {\sum_{x_M} \Psi(x_\Omega, x_M)},
\label{eq:marginal}
\end{equation}

where $M = \{1,\dots,n\} \setminus \Omega$ is the set of masked variables.

We implement two backends:
\begin{enumerate}[leftmargin=2em]
    \item {\bf Brute-force oracle}: Precompute all 288 valid $4\times4$ Sudoku solutions and filter by compatibility. This gives exact marginals for debugging.
    \item {\bf Variable elimination contraction}: Build factor tables $\psi_G$ for each constraint group, condition on observed variables by slicing, and compute marginals via sequential elimination of non-target variables. This uses numpy einsum for factor joining and achieves $\|q^C - q^{\text{BF}}\|_\infty < 10^{-14}$.
\end{enumerate}

\subsection{Denoisers}
\label{sec:denoisers}

We implement three denoisers:

\paragraph{Random denoiser.}
$p_i(v) = 1/d$ for all masked positions $i$. Baseline.

\paragraph{Local heuristic denoiser.}
For each masked $i$, forbidden values are those appearing in observed positions sharing a row, column, or box with $i$. The proposal is uniform over remaining values. This reasons only locally (within each constraint group) but does not capture global constraint interactions.

\paragraph{TN marginal denoiser.}
$p_i(v) = q_i(v)$ from the contraction or brute-force backend. This is the exact oracle, providing the logical posterior over masked variables given current observations.

\subsection{Product-of-experts correction}
\label{sec:poe}

Given a neural/noisy denoiser $p_i(v)$ and a TN marginal $q_i(v)$, define the combined distribution as in~\eqref{eq:poe}. For $\beta=0$ the model uses only logical inference; for $\beta=1$ only the neural/noisy denoiser; intermediate values interpolate.

We also define a {\em noisy denoiser} to simulate an imperfect neural model:

\begin{equation}
p_i^\sigma(v) = \operatorname{softmax}\bigl( \log(q_i(v)+\delta) + \sigma \,\xi_{i,v} \bigr)_v,
\label{eq:noisy}
\end{equation}

where $\xi_{i,v} \sim \mathcal N(0,1)$ and $\sigma \ge 0$ controls the noise level.

\subsection{Diffusion sampler}
\label{sec:sampler}

The iterative denoising loop proceeds as:

\begin{algorithmic}
\State $x^{(0)} \gets \text{corrupt}(x^{\text{true}}, \text{mask\_ratio})$
\For{$k = 0, \dots, K-1$}
    \State $\text{diag} \gets \text{Verifier}(x^{(k)})$
    \If{$\text{diag}.V = 0$ and $\forall i: x^{(k)}_i \neq \Mask$}
        \State \textbf{break}
    \EndIf
    \State $p \gets \text{Denoiser}(x^{(k)})$
    \If{TN correction}
        \State $q \gets \text{TNBackend}(x^{(k)})$
        \State $p \gets \text{PoE}(p, q, \beta)$
    \EndIf
    \State $U \gets \text{MaskPolicy}(x^{(k)}, p, \text{diag})$
    \For{$i \in U$}
        \State $x^{(k+1)}_i \gets \argmax_v p_i(v)$
    \EndFor
\EndFor
\end{algorithmic}

We use a confidence-based unmask policy: unmask $i$ if $\max_v p_i(v) \ge \tau$; if none meet the threshold, unmask the single most confident position to guarantee progress.

\section{Related work}
\label{sec:related}
% ==============================================================================

Masked diffusion language models~\cite{austin2021structured,chang2022masked,savinov2021step} generate discrete sequences by iterative unmasking. The mask policy and denoising schedule are active research topics~\cite{ou2024mdlm,gulrajani2024mdlm4lv}. Our work differs in introducing an external logical verifier and TN backend to guide both the denoising proposal and the mask selection.

Tensor-network methods for logical inference~\cite{glasser2020lattice,stoudenmire2016supervised} have been used for constraint satisfaction, probabilistic inference, and differentiable logic. Our work connects these to the masked diffusion framework.

Product-of-experts combination of neural and symbolic reasoning appears in neuro-symbolic AI~\cite{garcez2019neurosymbolic,manhaeve2018deepproblog}. Our PoE formulation is simple and testable, serving as a baseline for more sophisticated integration.

\section{Conclusion and next steps}
\label{sec:conclusion}
% ==============================================================================

We have built a working framework for tensor-guided masked diffusion on a finite symbolic domain. TN marginal denoising achieves 100\% completion in 1.4 steps, the contraction backend matches brute force to machine precision, and the architecture cleanly separates the denoiser, verifier, and mask policy concerns.

Immediate next steps include:
\begin{itemize}[leftmargin=2em]
    \item Complete the PoE beta/sigma sweep experiments;
    \item Add wrong-token corruption and verifier-guided remasking (Milestone 4);
    \item Implement a TT-based learnable mask policy (Milestone 5);
    \item Replace the synthetic denoiser with a small learned transformer (Milestone 6).
\end{itemize}
